Process

はじめての１からのホームページの作成だったのでこれからも再考しより良いものへ変えていきたいと考えている。

Concept

カッコよく、まとまりのある

自分の中でかっこいいとは、、、
大人らしい雰囲気の人。落ち着きがあってクール。落ち着きがあるとはどんなイメージを与えられているのか、大人と子供の反対の立場から考えた。

モノクロを主としてまとめていく上で、インパクトが欲しかったのでアクセントカラーを入れた。アクセントカラーは、自分の雰囲気に合うような色にしようと思ったので、色が与える印象を調べて決めた。

赤色が人に与える印象
赤色のイメージは、情熱 などの、燃え上がるようなイメージがある。自分の負けず嫌いで人と競い合うことに熱くなる、といったところが赤色にぴったりであると考えた。また、赤は一番に目を引く色なので、シンプルなデザインの中に入れる色としてはいいと思った。

黒色よりも赤色でアクセントを入れることで名前をより、注目的にする。　

UI Design

コンセプトをもとにデザインを設計。
コンセプトから、モノクロといったイメージになったのでこれを組み込んでいった。photoshopでのデザイン設計で、自分の中でのイメージを具現化した。ここでの設計をしっかりしておかないと、codingするときにデザインの大幅な変更は大変である。だから、デザイン設計はホームページを作る上で一番重要な作業といえる。

Coding

読みやすく、無駄を少なく
UIデザインを決定してから、コードに移す。細かい動きをjQueryで制御した。また、新しい作品をアップするときにより手間を省くことができるようなhtmlの構築をした。
