ACTEE

Arduino GUGEN 応用プロジェクト制作
制作期間：2016/09 ~ 2016/11 制作人数：3人 使用言語：C++(Arduino) GitHubページ

Concept

action × effection
よりアクティブに、より楽しく

アクティブな遊びにより効果的な反応がかえってくることでより遊びが楽しくなる。ACTEEは動きに合わせた反応を返してあげることでよりアクティブに、より楽しくなる遊びを実現するために作られたフライングディスクです。

Target user

大人から小さい子供までわいわい楽しめます。

Mode

モードは4つのモードがあり、好きなモードで遊ぶことができます。

Normal

ノーマルモードでは投げ方の良し悪しを判定し、色を変化させ、反応を返します。投げ方が良くないと赤に変化し、普通だと黄色、うまく投げられた時は青色に変化します。

Sugoroku

すごろくモードではノーマルモードで判断した投げ方の良し悪しから点数化し、その点数に合わせて赤いライトが点灯します。また赤く点灯するライトの数が最大になるとフラッシュライトになり、ポイントがリセットします。仲間内で誰が投げた時にフラッシュライトになったのか判定して遊ぶことができます。

Bomb

爆弾モードは爆弾ゲームのようなスリルをフリスビーで味わうことができるものです。上手く投げれないとその分早くACTEEの爆発（フラッシュ）がおこります。また、爆発までの時間はACTEE自体の色の変化から認識することができます。

Art

アートモードは動きに応じてライトの色が変化するモードです。素早く動かせば動かすだけ様々な色に変化し、ゆっくり動かすと色の変化も穏やかになります。自由に動かし光の変化を楽しむことと、一眼カメラの長時間露光モードかスマートフォンアプリで光の軌跡を撮影することで美しいアートを描くことができます。
